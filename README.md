# Sports-Injury-Predictions
This is a repository for our undergraduate honors thesis project to test the ability of deep learning to predict athlete injuries. By publishing this repository, we are sharing the data we extracted, our methods, model pipelines, and results.

Authors: Oscar Leeper and Sayalee Chivate
